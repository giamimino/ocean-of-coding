HTML-ის ძირითადი მაგალითები

HTML (ჰიპერტექსტის მარკირების ენა) არის სტანდარტული მარკირების
ენა, რომელიც გამოიყენება ვებ გვერდების სტრუქტურის შესაქმნელად.
აქ არის რამდენიმე ძირითადი HTML ელემენტი და კონცეფცია:
HTML დოკუმენტის სტრუქტურა:
HTML დოკუმენტი, როგორც წესი, სტრუქტურირებულია შემდეგი ელემენტებით:
ყველა HTML დოკუმენტი უნდა დაიწყოს დოკუმენტის ტიპის დეკლარირებით: <!DOCTYPE html > თავად HTML დოკუმენტი იწყება <html>-ით და მთავრდება </html>-ით. HTML დოკუმენტის ხილული ნაწილი არის <body> და </body> შორის.

<!DOCTYPE html>
<html lang ="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title >თქვენი გვერდის სათაური< /title>
< /head >
< body >
< შინარსი იწერება აქ >
< /body >
< /html >

  • < !DOCTYPE html > დეკლარაცია განსაზღვრავს, რომ ეს დოკუმენტი
    არის HTML5 დოკუმენტი
  • < html > ელემენტი არის HTML გვერდის ძირითადი ელემენტი
    ელემენტი
  • < head > შეიცავს მეტ ინფორმაციას HTML გვერდის შესახებ
    ელემენტი
  • < title > განსაზღვრავს სათაურს HTML გვერდისთვის
    (რომელიც ნაჩვენებია ბრაუზერის სათაურის
    ზოლში ან გვერდის ჩანართში) ელემენტი
  • < body > განსაზღვრავს დოკუმენტის სხეულს და წარმოადგენს
    კონტეინერს ყველა ხილული შინაარსისთვის, როგორიცაა
    სათაურები, აბზაცები, სურათები, ჰიპერბმულები, ცხრილები, სიები და ა.შ.

< !DOCTYPE > დეკლარაცია
< !DOCTYPE > დეკლარაცია წარმოადგენს დოკუმენტის ტიპს და
ეხმარება ბრაუზერებს ვებ გვერდების სწორად ჩვენებაში.
ის მხოლოდ ერთხელ უნდა გამოჩნდეს, გვერდის ზედა ნაწილში
(ნებისმიერი HTML ტეგების წინ).
< !DOCTYPE > დეკლარაცია არ არის რეგისტრირებული.
< !DOCTYPE > დეკლარაცია HTML5-ისთვის არის:
