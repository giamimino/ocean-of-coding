ინსტრუქცია

ისწავლეთ HTML Notepad-ის ან TextEdit-ის

ვებ გვერდების შექმნა და შეცვლა შესაძლებელია პროფესიონალური
HTML რედაქტორების გამოყენებით.

თუმცა, HTML-ის შესასწავლად ჩვენ გირჩევთ მარტივ ტექსტურ რედაქტორს,
როგორიცაა Notepad (PC) ან TextEdit (Mac).

ჩვენ გვჯერა, რომ მარტივი ტექსტური რედაქტორის გამოყენება კარგი გზაა
HTML-ის შესასწავლად.

მიჰყევით ქვემოთ მოცემულ ნაბიჯებს, რათა შექმნათ თქვენი პირველი ვებ გვერდი
Notepad-ით ან TextEdit-ით.

ნაბიჯი 1: გახსენით Notepad (PC) Windows 8 ან უფრო ახალი:

გახსენით საწყისი ეკრანი (ფანჯრის სიმბოლო თქვენი ეკრანის ქვედა
მარცხენა მხარეს). ჩაწერეთ Notepad.

Windows 7 ან უფრო ადრე:

Open Start > Programs > Accessories > Notepad

ნაბიჯი 1: გახსენით TextEdit (Mac)

გახსენით Finder > Applications > TextEdit

ასევე შეცვალეთ რამდენიმე პარამეტრი, რათა აპლიკაციამ სწორად შეინახოს ფაილები.
Preferences > Format > აირჩიე "Plain Text"

შემდეგ "გახსნა და შენახვა" განყოფილებაში მონიშნეთ ველი, რომელშიც ნათქვამია:
"HTML ფაილების ჩვენება HTML კოდის ნაცვლად ფორმატირებული ტექსტის ნაცვლად".

შემდეგ გახსენით ახალი დოკუმენტი კოდის დასაყენებლად.

ნაბიჯი 2: დაწერეთ ზოგიერთი HTML

ჩაწერეთ ან დააკოპირეთ შემდეგი HTML კოდი Notepad-ში:

ნაბიჯი 3: შეინახეთ HTML გვერდი

შეინახეთ ფაილი თქვენს კომპიუტერში.
აირჩიეთ ფაილი > შენახვა როგორც Notepad მენიუში.

დაასახელეთ ფაილი "index.htm" და
დააყენეთ კოდირება UTF-8-ზე
(რაც სასურველია HTML ფაილებისთვის).

ნაბიჯი 4: ნახეთ HTML გვერდი თქვენს ბრაუზერში

გახსენით შენახული HTML ფაილი თქვენს საყვარელ ბრაუზერში
(ორჯერ დააწკაპუნეთ ფაილზე, ან მარჯვენა ღილაკით - და აირჩიეთ
"გახსნა").
