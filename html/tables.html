HTML ცხრილები

HTML ცხრილები ვებ დეველოპერებს საშუალებას აძლევს, მოაწყონ
მონაცემები რიგებად და სვეტებად.

განსაზღვრეთ HTML ცხრილი

ცხრილი HTML-ში შედგება ცხრილის უჯრედებისგან რიგებისა და
სვეტების შიგნით.

მაგალითი:


< table border ="1px" >
  < tr >
    < th > სახელი < /th >
    < th > გვარი < /th >
    < th > ასაკი < /th >
  < /tr >
  < tr >
    < td > გიო < /td >
    < td > მიმინოშვილი < /td >
    < td > 15 < /td >
  < /tr >
  < tr >
    < td > ლუკა < /td >
    < td > ღუნაშვილი < /td >
    < td > 19 < /td >
  < /tr >
< /table >

ცხრილის უჯრედები

ცხრილის თითოეული უჯრედი განისაზღვრება < td > და < /td > ტეგით.



td ნიშნავს ცხრილის მონაცემებს.

ყველაფერი < td > და < /td > შორის არის ცხრილის უჯრედის შინაარსი.

მაგალითი:


< table border ="1px" >
  < tr >
    < td < > ემილია < /td >
    < td > ტობი < /td >
    < td > თეიმურაზი < /td >
  < /tr >
< /table >



შენიშვნა: ცხრილის უჯრედი შეიძლება შეიცავდეს ყველა სახის HTML
ელემენტს: ტექსტს, სურათებს, სიებს, ბმულებს, სხვა ცხრილებს და ა.შ.
ასევე ტეგი border არ უნდა გამოიყენოთ ცხრილებში რო დამატოთ
ხაზები css არის საჭირო მაგრამ თქვენ ეხლა გადიხარ html ის ტუტორიალებს
ამიტომ თქვენ არანაირი კავშირი არ გვაქვს css თან მომოვალში ისწავლით

ცხრილის რიგები

ცხრილის თითოეული მწკრივი იწყება < tr > -ით და მთავრდება < /tr > ტეგით.

tr ნიშნავს ცხრილის მწკრივს.


მაგალითი:


< table border ="1px" >
  < tr >
    < td > გიო < /td >
    < td > ნიკა < /td >
    < td > მელანო < /td >
  < /tr >
  < tr >
    < td > 16 < /td >
    < td > 14 < /td >
    < td > 10 < /td >
  < /tr >
< /table >
შედეგი:

გიო ნიკა მელანო
16 14 10


შეგიძლიათ იმდენი მწკრივი გქონდეთ, რამდენიც
გსურთ ცხრილში; უბრალოდ დარწმუნდით, რომ
უჯრედების რაოდენობა ერთნაირია თითოეულ რიგში.


შენიშვნა: არის შემთხვევები, როდესაც მწკრივს შეიძლება
ჰქონდეს ნაკლები ან მეტი უჯრედი, ვიდრე მეორეს. ამის
შესახებ მოგვიანებით თავში შეიტყობთ.


ცხრილის სათაურები

ზოგჯერ გსურთ, რომ თქვენი უჯრედები იყოს ცხრილის სათაურის უჯრედები. ასეთ შემთხვევებში გამოიყენეთ < th > ტეგი < td > ტეგის ნაცვლად:

th ნიშნავს ცხრილის სათაურს.


მაგალითი:


< table border ="1px" >
   < tr >
     < th > პირი 1 < /th >
     < th > პირი 2 < /th >
     < th > პირი 3 < /th >
   < /tr >
     < tr >
     < td > ნატო < /td >
     < td > საბა < /td >
     < td > თემო < /td >
   < /tr >
   < tr >
     < td > 16 < /td >
     < td > 14 < /td >
     < td > 10 < /td >
   < /tr >
< /table >
შედეგი:

პირი 1 პირი 2 პირი 3
ნატო საბა თემო
16 14 10

ნაგულისხმევად, ტექსტი < th > ელემენტებში არის თამამი და ცენტრში, მაგრამ
თქვენ შეგიძლიათ შეცვალოთ ეს CSS-ით.
